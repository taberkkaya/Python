## Requirements

The packages I used to run the code are listed in here.

>*Kodu çalıştırmak için kullandığım paketler burada listelenmiştir.*

**Install tkinter**

```
pip install tk
```

**Install pyshorteners**

```
pip install pyshorteners
```

**Install pygame**

```
pip install pygame
```

**Install pillow**

```
pip install pillow
```

**Install pytube**

```
pip install pytube
```

**Install pyttsx3**
```
pip install pyttsx3
```

**Install PyPDF2**
```
pip install PyPDF2
```

**Install instaloader**
```
pip install instaloader
```
